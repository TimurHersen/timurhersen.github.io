Kademlia vehicle routing system

This project was created by me and two other group members. The system simulates traffic in an environment where only automated vehicles exist, by using a node system based on DHT we can foresee and avoid traffic junctions.


Mapping system

This project was created by me and another member, it's a map system where locations are stored as nodes. By storing the information traversal times between nodes can by calculated by DFS and BFS algorithms.



Text summarization using GPT-4

This is the project that I'm currently working on with another team member, we are developing a tool which will be used to summarize academic literacy. Our goal is to increase the readability of these texts for people with reading disabilities. This project is built using a beta-key that was given by OpenAI.


Item registry

This is a small system for keeping track of inventory and auctioning items. The system consists of users, items and auctions, where each user can register and put up an auction for a specific item.
